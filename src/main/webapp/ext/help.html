1 Introduction

The cREST Client extension is a RESTful service client aiming to make the client's life as easy as possible. The UI is divided into two high level sections including the Request Builder, and Responses. The Request Builder includes sections for specifying a URI, an HTTP method, headers, and a request entity. The Responses section is where, well, your RESTful service responses are kept including a bunch of useful goodies that help you manage them.

2 Request Builder

The request builder contains all the ui components you need to use in order to submit an http request.
